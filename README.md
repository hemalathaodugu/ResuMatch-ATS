# ResuMatch ATS

## Resume Screening and Applicant Tracking System

# Project Overview

ResuMatch ATS is a web-based Resume Screening and Applicant Tracking System developed using Flask and MongoDB. The system helps users upload resumes, extract candidate information, calculate ATS scores, and manage candidate records efficiently.


# Features

* Upload resumes in PDF and DOCX formats
* Extract candidate details from resumes
* Calculate ATS scores based on matching criteria
* Search and manage candidate information
* Store candidate data using MongoDB
* Responsive and user-friendly interface
